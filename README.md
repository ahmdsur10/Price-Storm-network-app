# 🌊 نظام تحليل شبكات السيول
### GIS Flood Network Analysis System
**Developed by: Eng. Ahmed Adam**

---

تطبيق ويب متكامل لتحليل شبكات تصريف السيول، حساب الأطوال، وتقدير التكاليف.

## 🚀 المميزات
- 🗺️ خريطة تفاعلية لعرض شبكة السيول
- 📏 حساب تلقائي لطول كل خط (Haversine vectorized)
- 💰 حساب التكلفة لخط واحد أو أكثر
- ✏️ رسم خطوط جديدة وحساب تكلفتها
- 📊 جدول بيانات قابل للتصدير CSV
- 📁 يدعم GeoJSON و Shapefile (ZIP)

## 💡 الأسعار الإرشادية
| النوع | السعر (ريال/متر) |
|-------|-----------------|
| أنابيب بقطر 1400 ملم | 4,004 |
| قناة صندوقية (1.8 × 1.4) م | 9,336 |
| قناة مفتوحة (12 × 1.5) م | 13,052 |

## 🛠 تشغيل محلي
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📦 رفع Shapefile
اضغط مجلد الـ SHP كاملاً → ZIP → ارفعه في التطبيق
